# Crime Rate Prediction System

CrimePredict is a web-based crime rate prediction and analysis system that uses historical crime data and machine learning techniques to estimate future crime trends. The project provides an interactive dashboard for predicting crime rates and visualizing crime statistics through graphs and filters.

Users can analyze crime trends based on:

State
District
Crime Type

The system is built using Flask, Python, HTML, CSS, JavaScript, and machine learning libraries to deliver an easy-to-use and responsive crime analytics platform.
